Этот проект представляет из себя графический интерфейс для работы с базами данных, написанный на Python3.

Чтобы запустить приложение можно открыть build/exe.win-amd64-3.6/GUI_DB.exe

Сама папка build является сборкой кода и библиотек.
Чтобы посмотреть как делается сборка нужно открыть файл Create_exe.txt

Чтобы запустить оригинал, необходимо запустить файл GUI_DB.py в корне этой папки.
В файле DB.py содержится класс для работы непосредственно с базой данных с помощью sqlite3.
В файле GUI.py содержится класс с графичиским окном и графичискими элементами
В файле GUI_DB_template.py содежрится базовый шаблон для написания своего интерфейса.

В файлу AliasDatabases.txt содержатся имена и пути к базам данных. 
Если файл будет пуст, то будет предложено указать базу данных и она сохранится в этом файле.
В этом файле можно изменить название базы данных, но, желательно, больше ничего не изменять, поскольку формат хранящихся данных может повредиться и работа может быть некорректной.
Формат хранения записей: "Name = %path%/myDb.db"
Также, рекомендуется не переименовывать этот файл. Если он не будет найден, то будет предложено найти его


Этот графический интерфейс не закончен, поскольку у меня не хватает времени на это.
